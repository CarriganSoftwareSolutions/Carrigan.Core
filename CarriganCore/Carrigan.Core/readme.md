# Carrigan.Core

**Small set of helpers and extension methods for .NET (.NET 9).**
*Status: pre‑1.0.*

---

## Install

**.NET CLI**

```bash
dotnet add package Carrigan.Core
```

**Package Manager**

```powershell
Install-Package Carrigan.Core
```
