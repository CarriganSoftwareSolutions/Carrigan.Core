# Carrigan.Core.Mvc

**This is a small set helpers for ASP.NET Core MVC (.NET 9).**
*Status: pre‑1.0.*

---

## Install

**.NET CLI**

```bash
dotnet add package Carrigan.Core.Mvc
```

**Package Manager**

```powershell
Install-Package Carrigan.Core.Mvc
```

---


## Targets

* `net9.0`
